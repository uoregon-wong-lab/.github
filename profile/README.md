## Hi there 👋
This github organization is intended to store codes/programs used by the university of oregon wong lab.


Guidelines:

- Create all changes in a separate branch (not main), then merge when you are confident. 
- Only merge to main if you are the admin of the project or you have permission from the admin. 
- Be as thorough as you can in your repositories. Include READMEs, sample files with the correct formatting, custom packages, etc. You never know when a future grad student might have to figure out what it was you did. 


Please let me (Racheal) know if you have any questions! I hope this ends up being helpful!
